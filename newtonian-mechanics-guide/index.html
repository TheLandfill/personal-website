Below is the complete list of steps to solve a problem in Newtonian Mechanics. Optional steps have a checkbox that you can turn on or off to explore them further. Examples are provided below.

  1. Setup
    1. Determine the goal of the problem.
    2. Underline or write down any context-specific equations.
    3. Annotate the information with variable names or relevance to the goal.
  2. Interactions
    • Gravity: Every object with mass attracts every other object with mass.





    • Tension: Are there any ropes, chains, cords, cables, etc. that can be bent but not stretched or compressed?
    • Normal: Are there any objects that are in physical contact?
    • Dynamic Friction: Are there any objects that are currently sliding past each other?
    • Static Friction: Are there any objects that are in physical contact and could slide past each other?
    • Elastic/Spring Forces: Are there any springs, bungee cables, or similar objects that can stretch or compress without getting permanently deformed?
    • Buoyancy: Is there an object in a fluid in a gravitational field?
    • Constraint Forces: Is there anything preventing an object from moving in a direction other than the typical normal force?

  3. Systems: A system in this context means any collection of matter. Systems at this level are typically collections of rigid bodies, but you could, in principle, make a system with every left shoe in the world, the planet Jupiter, and the Andromeda Galaxy. Newton's Laws hold so long as you consider the center of mass of each system.

    For each question, write down the name of each prompt in the space below on a separate line.

    1. Rigid Bodies: Any object that does not significantly deform under stress.
    2. Isolated Systems: Any collection of objects that only interact with objects in the collection.
    3. Interaction Splitters: If there is an interaction you care about, an interaction splitter is a system that contains one of the objects in the interaction, but not both.
